# Projeto_Final_JS
incremento no exemplo passado em sala


Principais alterações feitas em JavaScript: 
  - Filtros por "Todas", "Pendentes", "Concluídas"
  - Validação para evitar tarefas duplicadas
  - Contador detalhado: "Total: X | Concluídas: Y | Pendentes: Z"

Principais alteracoes feitas em CSS:
  - Adicionado suporte para modo escuro
  - Detalhes em verde
  - Animações de entrada e saída para tarefas
  - Animações de entrada (slide + fade)
  - Animações de saída ao remover
